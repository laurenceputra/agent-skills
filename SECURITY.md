# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public issue.

Instead, email the maintainer at **security@agent-skills.example.com** with:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested fixes

We will acknowledge receipt within 72 hours and provide a remediation timeline.

## Supported Versions

This repository is maintained on a best-effort basis. Please use the latest version of the skills and installation script.
